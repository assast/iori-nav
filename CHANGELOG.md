# CHANGELOG

## 2026-05-25

- 将后台批量管理改为书签列表内联选择，并支持当前页批量删除、改分类和改隐私。
- 将分类管理改为表格展示，并通过排序值输入保存分类顺序。
- 修复分类排序保存与分类更新接口参数不匹配的问题。
- 将后台书签列表改为表格展示，并支持直接编辑排序值。
- 让书签更新接口支持局部更新，便于单独保存排序等字段。
